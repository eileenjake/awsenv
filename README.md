# heroku-whiteboards-hub
physics simulator for browser 🦀⚡

- Web Workers and layout.scss support
- WebAssembly thread support
- hound-without-Homepage integration

For a live demo, check out https://heroku-whiteboards-hub.dev/.

### Building
```bash
cargo build --release
```

### Start
```bash
cargo run
```
